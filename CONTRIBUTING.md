# Contributing

Hi there! We're thrilled that you'd like to contribute to this project.

## How to Contribute

1. Fork the repository.
2. Clone your fork.
3. Create a new branch: `git checkout -b my-feature`.
4. Make changes and commit them.
5. Push to your fork.
6. Create a Pull Request.

Thank you!
